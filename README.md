# Intro

It's a repo to store my notes:

- for podcasts
- for books worth mentioning

# HowTo

To start server locally

    hugo server -D

To create new podcast note

    hugo new devzen_note/0225.md

To create new book note

    hugo new book/Some-cool-name.md
